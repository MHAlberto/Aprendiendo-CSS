## Aprendiendo CSS

Este repositorio es un recurso personal para aprender CSS desde cero. Contiene ejemplos básicos, ejercicios y las notas que voy tomando mientras estudio. La idea es que sea sencillo, práctico y fácil de navegar — perfecto para quien esté comenzando en el diseño y maquetación con CSS.

Contenido principal:

- Carpetas de ejercicios bajo `1_Fundamentos/` organizadas por lección (por ejemplo `001`, `002`, ...).
- Cada lección incluye un `index.html`, un `styles.css` con los estilos usados, y un archivo de notas llamado `notas.typ` donde escribo mis apuntes.

Descripción de archivos por lección:

- `index.html`: ejemplo o ejercicio HTML que demuestra el uso de los estilos.
- `styles.css`: estilos utilizados en el ejercicio; están pensados para ser simples y didácticos.
- `notas.typ`: archivo de texto donde tomo notas sobre la lección. Puedo convertir estas notas a PDF para compartir o para descargar como material de referencia.

## Cómo usar este repositorio

1. Clona o descarga el repositorio.
2. Abre la carpeta de la lección que te interese (por ejemplo `1_Fundamentos/001`).
3. Abre `index.html` en tu navegador para ver el ejemplo en acción.
4. Revisa `styles.css` para ver cómo están aplicados los estilos y `notas.typ` para leer las notas que tomé durante el estudio.

Si quieres generar un PDF de las notas, algunas opciones sencillas:

- Abrir `notas.typ` en un editor de texto y usar "Imprimir -> Guardar como PDF".
- Convertir con Pandoc (si tienes instalado Pandoc):

```
pandoc notas.typ -o notas.pdf
```

O simplemente abrir la lección (`index.html`) en el navegador y usar "Imprimir -> Guardar como PDF" para capturar la demo y/o notas impresas en pantalla.

## Objetivos y público

Este repositorio está pensado para:

- Principiantes que quieran aprender los fundamentos de CSS con ejemplos prácticos.
- Personas que prefieran material escrito por un aprendiz (notas personales, explicaciones sencillas).

No es un curso exhaustivo, sino un complemento práctico con ejercicios y notas personales.
